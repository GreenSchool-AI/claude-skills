# Claude Setup Coach

A free Claude Skill by AI Courses by Green School that sets up your Claude so it stops losing details, making things up, and repeating the same mistakes in every new chat. It interviews you for a few minutes, then hands you ready-to-paste settings, a simple knowledge habit, and your own workflow skill.

## Who it's for

Anyone who uses Claude regularly and is tired of re-explaining themselves — professionals doing writing, knowledge work, data, or coding. No technical background required; it adapts to how technical you are.

## What you get

1. **Personal preferences** — one copy-paste block of durable rules that applies to every chat.
2. **A knowledge habit** — a couple of living files Claude reads before a task and updates after, so lessons stick.
3. **Your own workflow skill** — a tailored SKILL.md that enforces the routine for you.
4. **(Optional) Version control** — plain-language file history and private backups, with confirmation before anything is committed or pushed.

Plus a quick nudge to switch on Memory, use Projects, and turn on past-chat search.

## Why it works

Every chat starts fresh, so nothing carries over by itself. This skill moves your durable rules and hard-won lessons OUT of any single chat into places that persist — your preferences, a small knowledge base, and a workflow skill. Three common complaints, three fixes:

- **"It forgets what we discussed"** → Memory + Projects + living knowledge-base files.
- **"It makes things up"** → rules that tell Claude to ask instead of guessing, and to verify against a source instead of recalling.
- **"It repeats the same mistake in new chats"** → a recurring-issues file your workflow skill reads before it acts.

## How to use it (non-technical)

1. Open Claude with this skill added (see "Installing the skill").
2. Say something like "help me set up Claude so it stops forgetting things."
3. Answer a few short questions. Paste the preferences it gives you into your settings, save the knowledge files, and add the workflow skill it writes for you.

You don't need to name the skill — Claude recognises a setup / "stop forgetting" / "stop making things up" request.

## Installing the skill

Full per-platform steps are in the repository's [INSTALL.md](https://github.com/GreenSchool-AI/claude-skills/blob/main/INSTALL.md).

**The short version:**

1. **[⬇️ Download the skill (ZIP)](https://github.com/GreenSchool-AI/claude-skills/raw/main/skills/claude-setup-coach/claude-setup-coach.zip)** — one ready-to-upload file.
2. In Claude, turn on **Code execution and file creation** under **Settings → Capabilities** (one-time).
3. Go to **Customize → Skills**, click **"+" → Create skill → Upload a skill**, and upload the ZIP.
4. Toggle it on, start a new chat, and ask it to set you up.

## What it won't do

It won't change your settings for you — Claude can't edit your preferences or toggle Memory on your behalf, so it hands you the text and tells you exactly where it goes. And it won't push anything to the internet without asking.
